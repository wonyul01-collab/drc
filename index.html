Golden Rules

5가지 핵심 원칙

업무를 시작하기 전, 이 다섯 가지를 반드시 먼저 확인하세요.

📌
가격부터 정한다
목표 판매가격이 확정되어야 원가 구조, 노무 시간, 공정 영상, 계산서 금액이 모두 연동된다. 가격 없이 시작하면 모든 것이 뒤에서 흔들린다.
📖
지침을 먼저 읽는다
협회 고시·기준 지침·단체 표준을 제품 기획 시작 전 반드시 숙지. 추정이나 타사 제품 참고로 진행하면 중간에 전체가 틀어진다.
⏰
2주 전 완료가 기준
접수 기간 시작일 기준 2주 전에 모든 서류 1차 완료. 마감일에 몰리면 건보 서버 과부하로 업로드 오류가 반복 발생한다.
📦
앞에서 다 준비한다
서류, 자재 정보, 공정 시간은 모두 앞단에서 선행 정의. 뒤에서 물어보고 수정하는 구조를 없앤다. 뒷단은 확인과 업로드만 남겨야 한다.
✅
체크리스트로 검증한다
진행 여부를 기억이 아닌 체크리스트로 관리. 특히 변경·신규 추가 지침은 별도 이슈 리스트에 포함시켜 누락을 방지한다.

D-Day Process Map

D-Day 기준 전체 프로세스 맵

접수 시작일(D-Day)을 기준으로 각 단계를 역산한 타임라인입니다. 각 항목을 클릭하면 상세 내용을 확인할 수 있습니다.

사전 준비 단계 (D-90~D-30)
서류 작성 단계 (D-30~D-14)
마무리·검토 단계 (D-14~D-3)
접수 실행 단계 (D-3~D-2)
Phase 1 — 사전 준비 단계
D-90 이전
STEP 01 기준 지침 및 고시 스터디
협회 고시, 단체 표준, 건보 고시 확인 → 제품 구성 기준(수량·구성품), 포장 기준, 촬영 지침 파악
📄 문서 확인 ⚠️ 필수 선행
확인해야 할 항목
• 제품 1세트 구성 기준 (예: 팬티 1개 + 패드 몇 개가 1세트인지)
• 포장재 기준 (박스 사이즈, 재질, 표시 기재 사항)
• 공정 동영상 촬영 지침 (카메라 고정 여부, 시계 위치, 연속 촬영 기준)
• 신청 서류 목록 및 양식 (최신 버전 확인, 변경 사항 체크)

⚠️ 실패 사례: 시중 판매 제품을 보고 구성을 추정해 진행했다가 기준과 달라 전체 작업이 틀어진 사례 발생.
D-90 이전
STEP 02 목표 가격 책정 및 원가 구조 시뮬레이션
판매 목표가 설정 → 원부자재 단가, 노무비, 외주 가공비 역산 → 전체 원가 구조 확정
💰 가격 책정 ⚠️ 필수 1순위
가격 책정 방법
• 시장 조사 → 경쟁 제품 가격대 파악
• 최종 타깃 가격 설정 후 약 20% 버프를 얹어 신청 가격 결정
• 신청 가격 기준으로 원가 역산: 원부자재 단가 + 노무비(시간 × 단가) + 외주 가공비 + 포장재
• 이 시뮬레이션이 완료되어야 계산서, 명세서 금액이 확정됨

⚠️ 실패 사례: 가격 책정이 늦어져 공정 영상 시간을 뒤에서 강제 조정, 세금계산서 재발행하는 상황 반복 발생.
D-90 이전
STEP 03 접수 타임라인 수립 (역산)
접수 시작 예정일을 D-Day로 설정 → 각 단계별 완료 기한을 역산하여 내부 일정표 확정
📅 일정 관리
표준 일정 기준
• D-60: 자재 발주 및 비교 견적 착수
• D-45: 제품 샘플 완성 목표
• D-30: 모든 증빙 서류 1차 준비 완료
• D-14: 서류 전체 1차 완료
• D-7: 크로스 체크 및 보완 완료
• D-2~3: 온라인 등록
D-60
STEP 04 자재 발주 및 비교 견적 확보
자재 단가 상위 5개 항목 비교 견적 3개 업체 이상 확보 / 외주 가공비 비교 견적 3개 업체 확보 / 자재 발주 시 규격·재질·사양·사진 포함 명세서 요청
📄 증빙 서류 💰 비교 견적
비교 견적 기준
• 자재 단가 상위 5개 항목 각각 3개 업체 견적 필수
• 외주 가공비도 3개 업체 (당사 포함 총 3개)
• 자재 발주 시 업체에 요청할 내용: 상호명, 사업자등록번호, 재질, 규격, 사이즈, 사양, 자재 사진
• 거래명세서·세금계산서 수령 시 이 정보가 포함되도록 사전 룰 세팅

⚠️ 실패 사례: 외주 가공비 비교 견적 누락, 외주 가공비 자체 누락으로 사후 추가 준비하면서 전체 일정 지연.
D-45
STEP 05 제품 샘플 완성 및 사양 확정
제품 샘플 완성 → 실물 확인 (중량, 재질, 치수 실측) → 제품 설명서·특장점 내용 확정 → 포장 박스 사이즈·재질 확정
🔧 제품 준비 ⚠️ 마일스톤
샘플 확정 시 확인 항목
• 중량, 치수 직접 실측 (명세서 기재값과 일치 여부 확인)
• 재질 확인 (업체에 소재 증빙 요청)
• 제품 설명서 내용 최종 확정 (샘플 확인 후 변경사항 반영)
• 포장 박스 사이즈·재질 확정 (기준 지침의 포장 기준 준수 여부 확인)

⚠️ 실패 사례: 샘플이 늦게 도착해 실물 확인 결과 중량 차이 발생, 설명서·영상 전면 재작업.
D-60~30
STEP 06 판매 실적 관리 (2개월 전 판매 완료)
접수 예정월 기준 2개월 전까지 판매 실적 완료 / 판매 내역 정리도 동 시점에 완료하여 사전 준비 상태로 유지
📄 판매 실적
판매 내역서 관리 기준
• 접수 예정월 기준 전전달(2개월 전)까지 판매 실적 확보
• 판매 내역 엑셀 정리도 해당 시점에 사전 완료
• 기준 변경 여부를 접수 전 반드시 최신 고시로 재확인

⚠️ 주의: 기준이 과도기적으로 변경 중인 경우가 있으므로 반드시 최신 기준 확인 필수.
Phase 2 — 서류 작성 단계
D-45~30
STEP 07 공급계약서 작성 및 날인
공급계약서 양식 확인 → 작성 → 날인 → 보관 (2026년 상반기 신규 추가 서류)
📄 오프라인 서류
공급계약서 주의사항
• 2026년 상반기 신규 추가 — 양식을 최신 버전으로 반드시 확인
• 날인 일정 고려하여 충분한 시간 여유를 두고 준비
• 내용 오기재 시 재작성·재날인 필요하므로 초안 검토 후 날인
D-30
STEP 08 증빙 서류 전체 준비 완료
자재별 거래명세서·세금계산서 / 자재 사진 촬영 / 비교 견적서 3개 업체 / 외주 가공비 증빙 / 공급계약서 — 모든 오프라인 증빙 서류 완비
📄 증빙 서류 ⚠️ 마일스톤
필수 서류 목록
① 자재별 거래명세서 (자재별 각각)
② 자재별 세금계산서 (자재별 각각)
③ 비교 견적서 — 자재 단가 상위 5개 × 3개 업체
④ 비교 견적서 — 외주 가공비 × 3개 업체
⑤ 자재 사진 (자재별 각각)
⑥ 외주 가공비 세금계산서 및 거래명세서
⑦ 공급계약서 (날인 완료)
D-30~20
STEP 09 제조 공정도 작성 및 공정 시간 정의
가격 기준 노무비 역산 → 공정별 작업 시간 정의 → 제조 공정도 작성 → 촬영 시간 기준 확정
🔧 공정 관리
제조 공정도 작성 방법
• 책정된 가격 기준으로 노무비 총액 산출
• 공정별 작업 시간 배분 (합계가 노무비 계산 시간과 일치하도록)
• 제조 공정도 각 단계에 시간 기재
• 이 시간이 공정 동영상 촬영 기준이 됨 (촬영 시 맞춰서 진행)

💡 팁: 공정도 작성 시 각 공정별 인원수, 작업 방법, 필요 도구도 함께 기재해두면 영상 촬영 시 혼선이 없다.
D-25~20
STEP 10 공정 동영상 촬영 (지침 준수)
촬영 지침 사전 공유 → 담당자 숙지 확인 → 지침에 따라 촬영 → 촬영 후 지침 준수 여부 즉시 확인
🎬 동영상 ⚠️ 지침 준수
촬영 지침 핵심 체크
• 시계(초침 표시)는 화면 오른쪽 하단에 위치
• 카메라는 고정 상태 유지 (들고 다니며 촬영 금지)
• 작업 중 화면 중단 없이 연속 촬영
• 각 공정별 지정 시간에 맞춰 촬영
• 촬영 완료 후 즉시 영상 확인 → 시계 노출 여부, 카메라 고정 여부 체크

⚠️ 실패 사례: 시계 없이 촬영, 카메라 이동하며 촬영, 작업 중간에 끊고 재촬영 — 기준 미준수로 재촬영 가능성 발생.
D-25~15
STEP 11 재무 자료 준비 및 엑셀 입력
홈택스 PDF 다운로드 → PDF→엑셀 변환(AI·변환 프로그램 활용) → 엑셀 시트 입력 → 검증 함수로 오류 확인
💼 재무 자료
재무 자료 입력 방법
• 홈택스에서 PDF 형태로 다운로드
• PDF→엑셀 변환 프로그램 또는 AI(Claude 등)로 데이터 추출 → 오류 최소화
• 변환된 데이터를 엑셀에 붙여넣기 후 IF/EXACT 함수로 원본 대조 검증
• 수기 타이핑 입력은 지양 (오타 발생 가능성 높음)

💡 팁: 엑셀 시트를 미리 설계할 때 입력 셀 옆에 검증 컬럼을 만들어두면 입력 즉시 오류를 확인할 수 있다.
D-20~14
STEP 12 자재 현황표 및 전체 서류 입력
자재 현황표 입력 (품목명·재질·무게·사양·구입처·세금계산서 번호·단가·수량) / 제품 설명서 최종 확정 / 건보 엑셀 양식 전체 입력
📊 데이터 입력
자재 현황표 입력 항목
• 제품 품목명 / 재질 / 무게 / 사양
• 구입처 (상호명, 사업자등록번호)
• 세금계산서 번호 / 총 공급가액
• 최종 구매일 / 단가 / 수량

💡 팁: 자재 발주 시 업체에서 받은 명세서에 이 정보가 포함되도록 사전 요청해두면 입력 중 전화 확인 없이 바로 작업 가능.
Phase 3 — 마무리·검토 단계
D-14
STEP 13 전체 서류 1차 완료 기준점
모든 서류 및 입력 데이터 1차 완료 목표 — 이 시점 이후부터는 검토와 보완만 진행
✅ 마일스톤
D-14 기준점 체크리스트
□ 모든 오프라인 증빙 서류 완비

D-14~7
STEP 14 내부 크로스 체크
담당자 교차 검토 → 누락 항목 확인 → 지침 준수 여부 재검증 → 오류 수정
🔍 검토
크로스 체크 포인트
• 서류 누락 없는지 체크리스트 대조
• 입력 데이터 원본 대조 (세금계산서 번호, 공급가액 등)

• 지침 변경 사항 반영 여부 확인
D-7
STEP 15 최종 보완 완료 및 동결
보완 사항 반영 완료 → 이후 추가 수정 없이 동결 → 온라인 업로드 준비 상태 유지
✅ 동결
Phase 4 — 접수 실행 단계
D+2~3
STEP 16 건보 온라인 시스템 업로드 및 접수
접수 기간 시작 2~3일차에 업로드 (마감일 절대 금지) → 재무제표 직접 입력 → 파일 한 개씩 업로드 → 접수 확인
🌐 온라인 접수 ⚠️ 마감일 금지
온라인 업로드 주의사항
• 접수 기간 2~3일차에 등록 (마감일 등록 시 서버 과부하로 오류 반복)
• 재무제표: 시스템 내 직접 입력 항목 별도 존재 → 미리 데이터 준비
• 파일 업로드: 한 개씩 업로드하는 방식 → 파일 순서 미리 정리
• 업로드 중 오류 발생 시 건보 콜센터 연락 (임의 재시도 전 확인)
• 접수 완료 후 접수 번호 및 결과 화면 캡처 보관

⚠️ 실패 사례: 마감일에 서버 과부하로 업로드 오류 반복, 건보에서 필수 서류만 우선 제출 안내받은 상황 발생.

Detailed Guide

핵심 업무 상세 가이드

자주 실수하거나 처음 담당자가 놓치기 쉬운 항목을 상세하게 안내합니다.

💰
STEP 02
가격 책정과 원가 시뮬레이션

모든 서류의 기준점이 되는 핵심 작업입니다. 이 단계를 건너뛰거나 미루면 이후 모든 작업이 뒤에서 수정을 반복하게 됩니다.

  • 목표 판매가 설정: 시장 조사 후 타깃 가격 확정
  • 신청 가격: 타깃 가격 × 1.2 (약 20% 버프) 이상으로 설정
  • 원가 역산: 원부자재 + 노무비(시간 × 시간당 단가) + 외주 가공비 + 포장재
  • 계산서 금액: 원가 구조 확정 후 그에 맞게 발행
⚠️ 가격 확정 없이 계산서를 먼저 발행하면 나중에 전체를 다시 해야 합니다.
📄
STEP 04
비교 견적 준비 가이드

비교 견적은 자재와 외주 가공비 모두 3개 업체 이상이 필요합니다. 사전에 충분한 시간을 두고 업체를 섭외해야 합니다.

  • 자재 비교 견적: 단가 상위 5개 자재 × 3개 업체 견적 필수
  • 외주 가공비: 당사 포함 총 3개 업체 견적
  • 견적서 형식: 업체명, 사업자번호, 품목, 수량, 단가, 발행일 포함
  • 선정 근거: 비교 후 선정한 업체와 이유를 간략히 기재
💡 업체별 비교 견적서를 한 폴더에 정리하고 비교표를 엑셀로 만들어두면 서류 제출 시 편리합니다.
🎬
STEP 10
공정 동영상 촬영 체크리스트

촬영 지침을 숙지하지 않고 촬영하면 기준 미준수로 재촬영이 필요할 수 있습니다. 촬영 전 반드시 아래 항목을 확인하세요.

  • 시계 위치: 초침이 보이는 시계를 화면 오른쪽 하단에 고정 배치
  • 카메라: 고정 거치 상태에서 촬영 (이동하며 촬영 금지)
  • 연속 촬영: 작업 중간 끊김 없이 처음부터 끝까지 연속
  • 촬영 시간: 공정도에 기재된 시간에 맞춰 진행
  • 촬영 후 즉시 확인: 시계 노출 여부, 카메라 고정 여부 체크
⚠️ 작업자에게 왜 찍어야 하는지, 어떻게 찍어야 하는지 사전에 설명하고 협조를 구하세요.
💼
STEP 11
재무 자료 입력 자동화

홈택스에서 받은 PDF를 수기로 타이핑하면 오류가 발생합니다. AI 또는 변환 프로그램을 활용해 자동화하세요.

  • PDF 변환: PDF→엑셀 변환 프로그램 또는 Claude AI에 PDF 업로드 후 데이터 추출
  • 검증 방법: 입력 완료 후 엑셀에서 EXACT 함수로 원본과 대조
  • 검증 컬럼: 입력 데이터 셀 옆에 =IF(EXACT(입력셀, 기준셀), "OK", "오류") 형태로 설계
  • 오류 발생 시: 원본 확인 후 수정, 재검증
💡 엑셀 시트 템플릿을 미리 만들어두고 매번 재사용하면 작업 시간을 크게 줄일 수 있습니다.
🌐
STEP 16
건보 온라인 시스템 업로드

온라인 업로드는 마감일 절대 금지입니다. 서버 과부하로 업로드 오류가 반복되며 데이터가 유실될 수 있습니다.

  • 업로드 시기: 접수 기간 시작 2~3일차 (마감일 절대 금지)
  • 재무제표: 시스템 내 직접 입력 항목 → 데이터 미리 준비
  • 파일 업로드: 한 개씩 업로드 방식 → 파일 목록 순서 미리 정리
  • 오류 발생 시: 임의 재시도 전 건보 콜센터 문의
  • 완료 후: 접수 번호 및 결과 화면 캡처 저장
⚠️ 건보 시스템 로그인 상태는 일정 시간 후 자동 만료됩니다. 장시간 작업 시 주기적으로 저장하세요.
📦
신규 제품 전용
신규 제품 사전 세팅 가이드

기존 제품은 이전 기준을 준용하면 되지만, 신규 제품은 아래 항목을 반드시 사전에 확정해야 합니다.

  • 제품 구성 기준: 1세트 = 팬티 몇 개 + 패드 몇 개? 공식 지침 확인
  • 포장 기준: 박스 사이즈, 재질, 표시 사항 (지침 기준으로 확정)
  • 가격 구조: 신규 제품이므로 시장 조사부터 시작
  • 촬영 기준: 신규 공정이므로 지침 전체 재확인
  • 서류 양식: 최신 버전 확인 (변경 사항 이슈 리스트 작성)
⚠️ 기존 제품 기준을 그대로 적용하지 마세요. 신규 제품은 지침을 처음부터 새로 확인해야 합니다.

Lessons Learned

실제 실패 사례와 개선 방법

이번 업무를 통해 직접 경험한 실패 사례를 기록합니다. 같은 실수를 반복하지 않기 위한 기록입니다.

❌ 실패 사례 01
지침 없이 추정으로 시작
요실금 팬티 구성을 시중 제품 기준으로 추정해 진행. 나중에 정부 기준과 달라 전체 구성·가격·서류 전면 재작업.
✅ 개선 방법
지침 먼저, 추정 금지
시작 전 협회 고시·건보 기준 지침을 직접 확인하고 제품 구성 기준을 명확히 확정한 후 업무 시작.
❌ 실패 사례 02
외주 가공비 누락
고관절 보호대 패드 외주 생산 시 가공비 증빙 서류 및 비교 견적 모두 누락. 사후에 추가 준비하면서 전체 일정 지연.
✅ 개선 방법
외주 체크리스트 사전 확인
외주 발주 시점에 가공비 세금계산서, 거래명세서, 비교 견적서 3종을 체크리스트로 관리하여 누락 방지.
❌ 실패 사례 03
가격 책정 없이 계산서 발행
판매가를 확정하지 않은 상태로 계산서 먼저 발행. 나중에 가격 조정으로 계산서 재발행, 공정 시간 재조정 반복.
✅ 개선 방법
가격 책정이 1번
업무 시작 첫 번째 단계로 목표 가격 확정 → 원가 시뮬레이션 → 계산서 발행 순서를 원칙으로 고정.
❌ 실패 사례 04
공정 영상 촬영 지침 미숙지
시계 없이, 카메라 이동하며 촬영. 기준 미준수로 재촬영 가능성 발생. 작업자도 왜 찍어야 하는지 이해 못한 상태.
✅ 개선 방법
촬영 전 지침 공유 및 확인
촬영 전날 지침 내용을 담당자·작업자 모두에게 공유. 촬영 후 즉시 확인하여 재촬영 여부 현장에서 판단.
❌ 실패 사례 05
마감일 온라인 업로드
접수 마감일 업로드 시도 → 서버 과부하로 오류 반복 → 건보 통화 → 필수 서류만 우선 제출 안내 받음.
✅ 개선 방법
접수 2~3일차 업로드 원칙
모든 서류를 D-14까지 완료 → 크로스 체크 → 접수 시작 2~3일차에 업로드. 마감일 업로드는 절대 금지.
❌ 실패 사례 06
자재 사양 사후 확인
자재 현황표 작성 중 재질·사양 모르는 항목이 발생 → 업체 전화 확인 → 실물 측정 반복으로 작업 흐름 끊김.
✅ 개선 방법
발주 시 사양 포함 명세서 요청
자재 발주 시점에 업체에 재질·규격·사이즈·사진 포함 명세서를 요청하는 것을 발주 룰로 확정.
